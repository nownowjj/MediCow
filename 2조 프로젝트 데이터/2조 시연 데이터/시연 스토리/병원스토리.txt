1. 병원 로그인 페이지(hospital/members/login)
	1. 병원 회원 가입(hospital/members/new)
	1-1 : 회원가입 버튼을 클릭해 병원 회원가입 페이지로 이동합니다.
	1-2 : 회원가입이 완료되면 프론트 페이지로 이동합니다.
	
	2. 병원 로그인
	2-1 : 아이디와 비밀번호가 틀리면 오류 메세지를 발생시키고
	       성공적으로 로그인되면 (/hospital/main)으로 이동합니다.
2. 메인 페이지(hospital/main)
	1. 정보 확인
	1-1 : 현재 예약 환자의 수, 예약 대기 인원의 수, 운영시간, 병원 리뷰에 대한
	       정보를 확인할 수 있습니다.

	2. 공지사항 확인(/hospital/notice)
	2-1 : 공지사항을 확인할 수 있고, 공지사항 바로가기 버튼을 누르면
	       관리자 측에서 작성한 내용들을 조회할 수 있습니다.
	2-2 : 공지사항 제목 링크를 누르면 상세 내용을 확인할 수 있습니다.
	
	3. 문의사항 확인(/hospital/contact)
	3-1 : 문의사항을 확인할 수 있고, 문의사항 바로가기 버튼을 누르면
	       병원 측에서 작성한 내용들을 조회할 수 있습니다.
	3-2 : 문의사항 등록하기 버튼을 클릭하여 관리자에게 보내는 문의사항을
	       등록할 수 있습니다.
	3-3 : 문의사항 제목링크를 클릭하면 작성했던 문의사항을 수정, 삭제할 
	       수 있습니다.
3. 의사 명단 조회 페이지(hospital/doctors/lists)
	1. 의사 명단 조회
	1-1 : 현재 이 병원에 등록되어 있는 의사명단을 조회할 수 있습니다.
	
	2. 의사 명단 추가(/hospital/doctors/new)
	2-1 : 의사 명단 추가 버튼을 눌러 의사명단작성 페이지로 이동합니다.	
	2-2 : 의사 이름, 진료과목, 약력, 이미지를 업로드 한뒤 저장버튼을 눌러줍니다.
	2-3 : 의사 명단에서 조회가 가능합니다.

	3. 의사 정보 보기(/hospital/doctors/{1})
	3-1 : 등록된 의사의 이름, 진료과목, 약력을 조회할 수 있습니다.
4. 병원 정보 확인 페이지(hospital/my/info)
	1. 병원 정보 확인 페이지
	1-1 : 등록된 병원의 정보, 날짜 정보, 등록된 의사를 조회할 수 있습니다.
	1-2 : 병원 정보 확인하러가기 버튼을 클릭한 뒤 비밀번호를 입력하면
	     수정페이지로 이동할 수 있습니다.
	1-3 : 병원 운영시간 확인하러 가기 버튼을 클릭한 뒤 운영시간과 휴무일을
	     입력 할 수 있습니다. (수정도 가능합니다.)
	1-4 : 의사 명단 확인하러 가기 버튼을 누르면 위의 3. 2-1와 동일한 기능입니다 
	
	2. 병원 정보 확인하러가기(hospital/my/update/pwConfirm)
	
	3. 병원 운영시간 확인하러 가기(hospital/info/time)
	3-1 : 입력하러가기 버튼을 클릭하면 병원 운영 시간을 설정할 수 있는
	       페이지로 이동합니다.
	3-2 : 시간과 진료일, 휴무일을 설정할 수 있습니다.
