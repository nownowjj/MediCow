1. 메인 페이지(/user/)
	1 : (일반)로그인 버튼을 누르면 로그인 페이지로 이동합니다.

2. 로그인/회원가입 페이지(members/login)
	1 : REGISTER버튼을 누르면 회원 가입 페이지로 이동합니다.
3. 회원가입 페이지(members/join)
	1 : 이름과 이메일을 입력합니다.
	2 : 비밀번호와 비밀번호 확인을 입력합니다.
	3 : 다음Api이용해서 주소를 입력, 휴대폰번호, 나이, 성별을 입력합니다.
	4 : 가입이 완료되면 로그인/회원가입 페이지로 이동합니다.
	5 : 로그인을 완료하면 Main페이지로 이동합니다.
4. 메인 페이지
	1 : 본인의 아이디로 로그인하면 회원가입할 때 본인이 작성했던
	    주소가 검색창 아래에 뜹니다.
	2 : 원하는 진료과를 클릭 후 현재 위치 반경을 설정, 검색버튼을 클릭하면
	    위치 반경 토대로, 가까운 순서대로 이 웹사이트에 등록된 병원이 
	    검색됩니다.
	3 : 공지사항을 클릭하면 관리자 측에서 작성한 공지글을 조회 할 수 있습니다.
	4 : 건강정보를 클릭하면 건강에 관련된 Youtube 영상을 확인 할 수 있습니다.

5. 예약 페이지
	1 : 예약 내역 탭을 클릭하면 본인이 예약한 병원의 이름과 예약상태, 리뷰등 
	     조회가 됩니다.
	2 : 병원 이름을 클릭하면 병원 상세페이지를 조회할 수 있습니다. 
6. 마이 페이지
	1 : 내 정보 수정버튼을 클릭하면 비밀번호를 입력한 뒤 개인 정보를 수정할
	    수 있습니다.
